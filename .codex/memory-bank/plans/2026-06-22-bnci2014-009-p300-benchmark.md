# BNCI2014_009 P300 Benchmark Run

Status: in_progress
Last updated: 2026-06-23
Next stage: 7 - Final Notebook, Report, And Memory Update (awaiting review)

## Goal

Build a leakage-aware, reproducible benchmark for BNCI2014_009 P300 single-epoch
`Target` vs `NonTarget` classification across classical EEG/ERP methods and the project's deep
learning model families.

## Scope

Included:

- MOABB loading for `BNCI2014_009`.
- Binary P300 epoch classification: `Target` vs `NonTarget`.
- Primary protocol: subject-wise generalization with no subject overlap between train and test.
- Secondary protocol only after Stage 1 confirms a clean session/trial contract.
- Classical baselines:
  - dummy/majority baseline;
  - ERP amplitude/vector features plus shrinkage LDA;
  - ERP features plus Logistic Regression;
  - ERP features plus Linear SVM;
  - ERP features plus Ridge Classifier;
  - xDAWN plus LDA or Logistic Regression;
  - xDAWN covariance or tangent-space Riemannian pipeline where `pyriemann` supports the target
    shape and runtime.
- Deep-learning baselines:
  - raw ERP CNN baseline;
  - EEGNet;
  - DeepConvNet;
  - ShallowConvNet;
  - optional exploratory EEGNet variants if the input contract is stable.
- Exploratory spectral deep-learning variants only after raw ERP models are working.
- Artifacts under `artifacts/experiments/bnci2014_009/`.
- Executed notebooks under `notebooks/`.

Excluded:

- State-of-the-art claims.
- Character-level spelling accuracy until trial, sequence, and stimulus semantics are audited.
- Treating CSP as a primary P300 model; CSP may be used only as an optional negative/control
  comparison after P300-appropriate baselines exist.
- Any learned preprocessing or threshold fitted outside training folds.

## Acceptance Criteria

- Dataset adapter emits deterministic rows with subject, session, run, epoch, event label, sampling
  frequency, channel metadata, and a stable sample key.
- Splits are audited for subject and sample-key leakage.
- Train/test class presence and class imbalance are reported for every fold.
- All scalers, feature selectors, xDAWN transforms, covariance estimators, thresholds, and neural
  normalizers are fitted only on training data within each fold.
- Classical and deep-learning runs produce immutable artifacts with config, environment, split,
  metrics, and inventory payloads.
- Metrics include balanced accuracy, macro F1, target recall, confusion matrix, ROC-AUC, and PR-AUC
  when scores are available.
- The final executed notebook compares all completed model families with fold-level summaries and
  limitations.
- Focused tests, `uv run ruff check .`, and `git diff --check` pass for implemented stages.

## Decisions And Assumptions

- Primary scientific claim target is cross-subject single-epoch P300 target detection.
- Balanced accuracy is the primary classification metric because `NonTarget` epochs are expected to
  dominate.
- ROC-AUC and PR-AUC are reported when continuous scores are available.
- Raw ERP/time-domain representations are the primary neural input; spectral representations are
  exploratory for P300.

  subjects, 16 EEG channels, and 256 Hz source sampling.
